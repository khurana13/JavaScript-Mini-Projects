# Weather Station

A simple and beginner-friendly weather application built using **HTML, Bootstrap, JavaScript, and OpenWeather API**.

This project allows users to enter a city name and fetch current weather details such as **temperature, weather condition, humidity, wind speed, and weather icon**.

---

## 📌 Features

- Search weather by city name
- Display current temperature in Celsius
- Show weather condition
- Display humidity percentage
- Display wind speed
- Show weather icon
- Simple and clean Bootstrap-based user interface

---

## 🛠️ Tech Stack

- HTML
- Bootstrap 5
- JavaScript
- OpenWeather API
